# Crevia Container and Environment Pack One

Container and Environment Pack One is an authoring-only content pack for container network pressure, environmental care, visible service, and district trust recovery.
It strengthens Cumhuriyet, Yesilvadi, and Merkez while adding container and environment support families for Sanayi and Istasyon.

## Scope

- 5 districts: Merkez, Cumhuriyet, Sanayi, Istasyon, Yesilvadi.
- 16 event family entries: Cumhuriyet 5, Yesilvadi 5, Merkez 3, Sanayi 2, Istasyon 1.
- 64+ variant copy entries, at least 4 per family.
- 6 echo surfaces per family: advisor, report, social, map, tomorrow preview, result.
- Container network, environment care, container pressure, and resource fatigue intents are metadata/content links only.
- Per-family hints: container network, environment care, field, follow-up.

## District Weight

Cumhuriyet families focus on night street residue, block-front overflow, merchant complaints, social trust repair, and site-entry recovery windows.

Yesilvadi families focus on park-edge container balance, quiet collection plans, green corridor care, post-picnic traces, and calm visible service tone.

Merkez families focus on square-edge visibility, dense passage container lines, citizen-visible response, and container network follow-up.

## Variant Coverage

Variant kinds include normal, improved, worsened, carry over, reward, comeback, resource fatigue, district trust, controlled crisis-adjacent, and recovery.
Operation era context stays in hints and metadata only, not as primary event framing.

## Container, Environment, and Fatigue Intent Coverage

Every family includes `containerNetworkIntent` and `containerPressureIntent`.
Every family includes `environmentCareIntent`; Merkez families also include `visibleServiceIntent`.
`resourceFatigueIntent` is present on all families for authoring linkage to resource visuals.

## Echo Surfaces

Each family ships advisor, report, social, map, tomorrow preview, and result echoes.
Container and environment short hints are stored as map_hint copy blocks and pack metadata.

## Runtime Boundary

- Runtime activation yok.
- Event generation rewrite yok.
- Persist shape degismez.
- SAVE_VERSION degismez; it remains 23.
- Container network engine, district operation action engine, UI, analytics, and IAP degismez.

## QA

Run:

```bash
npm run verify:container-environment-pack-one
```

The verifier checks district distribution, variant count, echo completeness, hint coverage, duplicate risk (internal and vs District Pack One and Vehicle Route Pack One), mobile copy length, controlled crisis-adjacent tone, dependent pipeline scenarios, and SAVE_VERSION.
